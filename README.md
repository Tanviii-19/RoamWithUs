# RoamWithUs ‚Äì Full Stack Travel Booking Application

RoamWithUs is a **full-stack web application** designed to explore and book travel destinations in **Uttarakhand, India**.  
The project demonstrates end-to-end development using modern frontend and backend technologies with role-based access.

---

## Tech Stack

### Frontend
- React.js
- Tailwind CSS
- React Router DOM
- Lazy loading & UI animations

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose (ODM)

---

## Features

### User
- Auto-sliding informational section (About, Tips, Best Time, etc.)
- Book travel packages after login
- Responsive and animated UI

### Admin
- View all bookings
- Delete bookings
- Restricted admin dashboard access
